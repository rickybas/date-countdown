## Dependencies
These dependencies must be present before building

- `valac>= 0.22`
- `meson>=0.40.1`
- `gtk+-3.0`
- `granite`

## Build

```sh
git clone https://github.com/rickybas/date-countdown.git
cd date-countdown
meson build
cd build
ninja
```

## Installation

```sh
sudo ninja install
./com.github.rickybas.date-countdown
```
